Tanee Revenge

Unity C#2D ArcadeTeam Project

Genre: Beat'em up, Arcade, Action

I'm working on game design, level design, and programming. My learning focused on building a state-machine system, developing enemy combat behavior, and designing the wave progression for levels.

As times have changed, urban development has rapidly expanded. Ruthless capitalists have invaded the banana groves, unaware that this land is home to a large community of Nang Tani (banana tree spirits). This sparks a battle as the Tani rise up to protect their homeland from the greedy invaders.

Control the action in a 2D side-scrolling world, using left and right movements. Defeat enemies with powerful attacks in wave after wave.

Features

  • State Machine — Basic state machine handling core game flow (Start → Play → Game Over).
  • Wave Progression — Progressive enemy waves, with increasing number and types of enemies.
  • Level Systems — Modular level pieces and spawn systems for quick iteration of encounters.
