Game Jam: God Builder

Unity 3D Jam Team Project 2019

Jam Site: https://globalgamejam.org/2019/games/god-builder

Genre: Tower stacking

Recreated a classic moblie game Stack Builder: Skyscraper from memory, made in 48 hours for the Global Game Jam 2019 site in Chiang Mai.

This game is about two gods trying to build a home for their people — but there can only be one home. Only one god can win.
